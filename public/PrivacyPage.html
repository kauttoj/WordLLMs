Privacy Policy

This Privacy Policy describes how WordLLMs handles your data.

No Data Collection

WordLLMs does not collect, track, store, or transmit any user data to us or to any third-party analytics or tracking service. There is no telemetry, no usage analytics, and no tracking of any kind built into this application.

Where Your Data Goes

All data processing happens locally on your machine, with one important exception:

  • AI Provider Requests: When you send a message, your input is sent directly to the AI provider you have configured (e.g., OpenAI, Azure OpenAI, Google Gemini, Groq, Ollama, LMStudio). This communication is between your machine and the provider's API — we are not involved and have no access to these requests or responses.

For any sensitive data, we recommend either using local LLM or setting up secure cloud account in Azure.

Local Storage

Your settings, API keys, and conversation history are stored locally in your browser's localStorage and IndexedDB. This data never leaves your machine unless you explicitly send it to an AI provider as part of a conversation.

API Keys

API keys you enter are stored only in your browser's localStorage. They are sent directly to the respective AI provider when making requests. We never receive, store, or have access to your API keys.

Third-Party Services

If you enable web search functionality, search queries are sent to the Tavily Search API using your own API key. This is an optional feature that you control entirely.
